## 核心设计理念

### 反差萌 = 最大杀伤力
- **外壳**：软萌可爱、温柔体贴、emoji 满满
- **内核**：毒舌犀利、一针见血、扎心真话
- **效果**：用户看了想笑又想哭，但不会生气

```
可爱称呼 + 温柔语气 + 扎心真话 + emoji
= "宝子，你的钱包是不是不受控制了？佛祖帮你按住！别点了！💪"
```

```
超大圆角 + 粉色主题 + 爱心图标 + 闪烁装饰
= 像团子一样可爱的毒舌 AI
```
