# ⛩️ Gas Temple - 功德无量

> "既然都要亏，不如亏得好笑一点"

Gas Temple 是一个加密货币研究娱乐平台，将枯燥的加密货币研究转化为"抽卡式"游戏体验。

## 🎮 核心玩法

- **盲盒投研** - 随机抽取加密项目卡牌，获得极简研报
- **庞氏指数** - 直观展示项目风险等级
- **韭菜名人堂** - 链上排行榜记录"光辉战绩"
- **SBT勋章** - 灵魂绑定的"耻辱勋章"系统

## 🛠️ 技术栈

- **框架**: React 18 + TypeScript
- **构建**: Vite
- **样式**: TailwindCSS
- **动画**: Framer Motion
- **状态**: Zustand
- **Web3**: 
  - Solana: @solana/wallet-adapter
  - BNB Chain: ethers.js
