**Demo 模式现在无需连接钱包即可使用！**

只需在 URL 中添加 `?demo=` 参数，然后点击"💰 提现 $GONGDE"按钮即可查看不同等级。

## 📱 操作流程

1. **打开浏览器**，输入带有 `?demo=` 参数的 URL
2. **点击页面上的"💰 提现 $GONGDE"按钮**
3. **查看弹窗**，应该看到：
   - 🎬 Demo 模式横幅
   - 对应等级的 SKR 持仓
   - DEMO 徽章
4. **输入提现金额**（例如 1000）
5. **查看计算结果**
6. **关闭弹窗**，修改 URL 参数测试其他等级

## 🔄 切换等级

**方法 1：修改 URL**
- 直接在浏览器地址栏修改 `?demo=tourist` 为 `?demo=monk`
- 按回车刷新页面
- 重新打开提现弹窗

**方法 2：使用书签**
- 为每个等级创建书签
- 快速切换测试

## 🐛 故障排除

### 问题：看不到 Demo 模式横幅
**解决方案：**
1. 检查 URL 是否包含 `?demo=` 参数
2. 刷新页面（Ctrl+R 或 F5）
3. 清除浏览器缓存

### 问题：等级不对
**解决方案：**
1. 检查 URL 参数拼写（tourist/pilgrim/monk/abbot）
2. 确保参数小写
3. 关闭提现弹窗，刷新页面，重新打开

### 问题：计算结果不对
**解决方案：**
1. 这是正常的！每个等级的税率不同
2. 参考上面的"预期结果"检查
