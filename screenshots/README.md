# 应用截图目录

此目录用于存放Gas Temple应用截图，用于提交到Solana Mobile dApp Store。

## 要求
- 至少4张截图
- 尺寸：1080x1920像素（竖屏）
- 格式：PNG或JPEG
- 每张文件大小：<2MB

## 建议截图内容

### screenshot1.png - 首页/抽卡界面
展示抽卡动画和项目卡片，突出"开始抽卡"按钮。

### screenshot2.png - 项目详情页
展示加密项目详细分析，显示风险等级和社区评价。

### screenshot3.png - 排行榜页面
展示"韭菜名人堂"排行榜，显示用户排名和战绩。

### screenshot4.png - 钱包连接界面
展示钱包连接流程，显示Solana和BNB Chain切换。

## 生成方法
1. 在Chrome开发者工具中设置设备模拟（1080x1920）
2. 导航到不同页面并截图
3. 保存为PNG格式到此目录
4. 更新`public/manifest.json`中的截图URL
